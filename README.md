# Проект: Сайт "Вкусные истории" с админ панелью для управления каталогом ресторана.
## <img width="1904" height="912" alt="image" src="https://github.com/user-attachments/assets/4534cf80-5224-4a7e-ab50-041094d3b730" />
Данный сайт предназначен для знакомтва клиентов с самим рестораном, а также его меню. На сайте имеется страница лендинг (главная), а такжестраница меню, где клиенты уже могут ознакомиться с меню ресторана, в котром расписаны цены и названия, и подробное описание предоставляемого ассортимента.

## Основная структура сайта:
* `index.php` - главная страница сайта с краткой информацией о ресторане и карусель-товаров.
* `catalog.php` - страница сайта, на котором показан весь каталог ретсорана с админ-панелью для управления.
* `cart.php` - карта сайта
* `reg.php` - страница регистрации пользователей
* `authoriz.php` - страница авторизации пользователей

## Админ панель
У администратора появляется меню, которое дает возможность добавить новый товар, категорию, также администратор может редактровать карточки товаров уже существующих позиций.

## Тенхологический стек
* Backend: PHP (простая логика без фреймворков)
* База данных: MySQL (подключение через `connect.php` используя метод mysqli)
* Frontend: HTML5, CSS3 ( `CSS/` с использованием библиотеки bootstrap)
* Клиентские скрипты: javascript
