# Проект: Сайт «Вкусные истории» с админ-панелью для управления каталогом ресторана

## <img width="1904" height="912" alt="image" src="https://github.com/user-attachments/assets/4534cf80-5224-4a7e-ab50-041094d3b730" />

Данный сайт предназначен для знакомства клиентов с рестораном и его ассортиментом. На ресурсе реализована главная страница (лендинг) и страница каталога, где посетители могут изучить меню: названия блюд, цены и подробные описания товаров.

## Основная структура сайта
* `index.php` — главная страница сайта с краткой информацией о ресторане и каруселью товаров.
* `catalog.php` — страница со всем каталогом ресторана и интерфейсом админ-панели.
* `cart.php` — корзина / карта сайта.
* `reg.php` — страница регистрации пользователей.
* `authoriz.php` — страница авторизации пользователей.

## Админ-панель
Администратору доступен функционал для управления контентом, который позволяет:
* Добавлять новые товары и категории.
* Редактировать карточки уже существующих позиций.
* Управлять актуальностью меню в режиме реального времени.

## Технологический стек
* **Backend:** PHP (чистая логика без использования фреймворков).
* **База данных:** MySQL (подключение через `connect.php` с использованием расширения `mysqli`).
* **Frontend:** HTML5, CSS3 (верстка с использованием библиотеки Bootstrap).
* **Клиентские скрипты:** JavaScript.
